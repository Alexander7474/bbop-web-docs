\doxysubsection{Detailed Description}
Class parente de toutes les formes géométriques(\+Rectangle\+Shape, Convex\+Shape, Circle\+Shape, Sprite) 

Elle stock et gère tous les attribut de base d\textquotesingle{}une forme géométrique comme sa position ou encore sa couleur

\begin{DoxyAttention}{Attention}
Certaine \doxylink{classShape}{Shape} utilise des attributs de cette class parent de manière différente donc renseignez vous sur les shapes que vous utilisez 
\end{DoxyAttention}
